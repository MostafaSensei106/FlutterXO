# torikkusu

"torikkusu"is a simple yet exciting strategic game. The game requires two players, each drawing either a circle or a cross in the squares of a 3x3 grid. The goal of the game is to have three of your symbol in a row, whether it be vertically, horizontally, or diagonally, before your opponent. This game requires strategic planning and quick thinking.
